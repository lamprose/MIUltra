## MIUltra
一个可能使MIUI更好用的xposed模块

### 使用条件
+ MIUI12.5
+ Android11(其他版本自测)
+ [LSPosed](https://github.com/LSPosed/LSPosed)

### 使用前须知
本模块部分功能开启后可能导致一些不可预估的问题,请了解相关功能后开启.开启则表明你已熟知相关后果且由本模块不可预估的后果需由你自行承担.

### 功能

- [x] 状态栏模糊风格修改(包括模糊半径)  
- [x] 非默认主题使用模糊  
- [x] 状态栏图标部分强制原生风格
- [x] 状态栏首条通知默认折叠
- [x] 桌面抽屉模式顶部分类删除全部分类
- [x] 桌面抽屉模式分页内删除编辑
- [x] 屏蔽云控动态FPS
- [ ] 更多.....
